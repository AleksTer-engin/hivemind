## 📦 Сервисы

| Сервис | Язык | Описание |
|--------|------|----------|
| **api-gateway** | go | Входная точка API, маршрутизация запросов к сервисам |
| **classifier** | python | Классификация текстов, определение тегов |
| **embedder** | python | Создание векторных эмбеддингов для текстов |
| **hivemind-ui** | python | Веб-интерфейс для взаимодействия с HiveMind |
| **linker** | go | Установление связей между документами на основе эмбеддингов и метаданных |
